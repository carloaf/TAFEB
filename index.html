Política de Privacidade

O aplicativo foi desenvolvido para auxiliar na consulta e pesquisa de índices de desempenho físico com base em tabelas detalhadas. Ele permite uma pesquisa personalizada de acordo com a idade, gênero e tipo de exercício, abrangendo modalidades como corrida, flexões na barra, abdominais e flexão de braços. A ferramenta é ideal para quem deseja monitorar e planejar seus treinamentos físicos, fornecendo informações úteis para otimizar seu desempenho.

Coleta e Uso de Informações

O aplicativo coleta apenas as informações necessárias para que o usuário possa realizar pesquisas personalizadas de acordo com sua idade, gênero, modalidade de exercício (corrida, flexões na barra fixa, abdominais supra e flexão de braços) e Linha de Ensino.

Armazenamento de Dados

O aplicativo utiliza um banco de dados local SQLite para armazenar as tabelas de desempenho físico. Isso garante que o usuário tenha acesso às informações, mesmo sem conexão com a internet. Nenhum dado pessoal ou resultado de pesquisa é compartilhado com terceiros ou armazenado em servidores externos.

Acesso Offline

Uma das principais vantagens do aplicativo é a possibilidade de acesso offline. As tabelas de índices físicos e os requisitos exigidos são armazenados localmente, permitindo que os usuários façam consultas a qualquer momento, sem necessidade de conexão à internet.

Segurança dos Dados

Todos os dados armazenados no dispositivo são protegidos pelo próprio sistema de segurança do Android. O aplicativo não coleta ou armazena dados pessoais sensíveis, como informações de contato ou localização.

Compartilhamento de Dados

Este aplicativo não compartilha informações com terceiros. Todos os dados manipulados são utilizados exclusivamente dentro do contexto do próprio aplicativo para oferecer resultados precisos de acordo com os critérios inseridos pelos usuários.

Alterações na Política de Privacidade

Reservamos o direito de alterar esta política de privacidade a qualquer momento. Qualquer alteração será comunicada através de uma atualização desta página. Recomenda-se que os usuários verifiquem esta política periodicamente.
